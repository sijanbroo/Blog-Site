## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

```bash
git clone https://github.com/sijanbroo/blog-site.git
cd blog-site
npm install
```

### Running the Project

```bash
npm start
```

The application will be available at `http://localhost:3000`
